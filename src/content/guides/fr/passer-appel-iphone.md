## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a passer un appel telephonique sur votre iPhone. Vous apprendrez a appeler quelqu'un depuis vos contacts et a composer un numero de telephone directement.

## Ce dont vous avez besoin

- Un iPhone avec un forfait telephonique actif ou une carte SIM

## Les etapes

### Etape 1 : Ouvrir l'application Telephone

Trouvez l'application **Telephone** sur votre ecran d'accueil. Elle a une icone verte avec un combine blanc. Appuyez dessus.

### Etape 2 : Appuyer sur l'onglet Clavier

En bas de l'ecran, vous verrez plusieurs onglets. Appuyez sur **Clavier**. Cela affiche un pave numerique ou vous pouvez taper un numero de telephone.

### Etape 3 : Composer le numero

Utilisez le pave numerique pour taper le numero de telephone que vous voulez appeler. N'oubliez pas l'indicatif regional. Si vous appelez un autre pays, commencez par l'indicatif de pays (par exemple, +33 pour la France).

### Etape 4 : Appuyer sur le bouton d'appel vert

Une fois le numero complet entre, appuyez sur le **bouton telephone vert** en bas du clavier. Votre iPhone commencera a appeler ce numero.

### Etape 5 : Terminer l'appel

Quand vous avez fini de parler, appuyez sur le **bouton rouge** sur votre ecran pour raccrocher. L'appel se terminera immediatement.

## Conseils utiles

- **Appeler depuis les Contacts** : Au lieu de taper un numero, appuyez sur **Contacts** en bas de l'application Telephone. Trouvez la personne que vous voulez appeler et appuyez sur son nom, puis sur son numero de telephone.
- **Appeler depuis les Recents** : Appuyez sur **Recents** en bas pour voir les numeros que vous avez appeles ou recus recemment. Appuyez sur n'importe quel numero pour le rappeler.
- **Appels d'urgence** : Vous pouvez toujours composer le numero d'urgence local (comme le 112 en Europe ou le 15/17/18 en France) meme sans carte SIM.
