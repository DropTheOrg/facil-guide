## Ce que vous allez apprendre

Dans ce guide, vous apprendrez a supprimer des applications de votre iPhone. Retirer les applications que vous n'utilisez plus libere de l'espace de stockage et garde votre ecran d'accueil bien range.

## Ce dont vous avez besoin

- Un iPhone

## Les etapes

### Etape 1 : Trouver l'application a supprimer

Regardez votre ecran d'accueil et trouvez l'application que vous souhaitez retirer. Si vous avez beaucoup d'applications, balayez vers la gauche pour parcourir les pages ou balayez vers le bas et utilisez la **barre de recherche**.

### Etape 2 : Appuyer longuement sur l'icone

Placez votre doigt sur l'icone de l'application et **maintenez-le appuye** pendant environ deux secondes. N'appuyez pas trop fort. Un petit menu apparaitra a l'ecran.

### Etape 3 : Appuyer sur Supprimer l'app

Dans le menu qui apparait, appuyez sur **"Supprimer l'app"**. Cela vous donnera des options pour ce que vous voulez faire avec l'application.

### Etape 4 : Appuyer sur Supprimer l'app (confirmation)

Un nouveau menu apparait vous demandant si vous voulez retirer l'app de l'ecran d'accueil ou la supprimer entierement. Appuyez sur **"Supprimer l'app"** pour la retirer completement de votre iPhone et liberer l'espace de stockage qu'elle utilisait.

### Etape 5 : Confirmer la suppression

Un message de confirmation apparait pour vous demander si vous etes sur. Appuyez sur **"Supprimer"** pour confirmer. L'application sera retiree de votre iPhone. L'icone disparaitra de votre ecran d'accueil.

## Conseils utiles

- **Verifier le stockage** : Allez dans **Reglages** puis **General** puis **Stockage iPhone** pour voir quelles applications prennent le plus de place. Supprimez d'abord les plus volumineuses.
- **Decharger les apps inutilisees** : Dans les memes reglages de stockage, vous pouvez activer **"Decharger les apps inutilisees"** qui retire automatiquement les applications que vous utilisez rarement tout en conservant leurs donnees.
- **Reorganiser les apps** : Si vous voulez simplement deplacer des applications sans les supprimer, appuyez longuement sur une app puis appuyez sur **"Modifier l'ecran d'accueil"**. Vous pouvez ensuite glisser les apps vers de nouvelles positions.
- **Bibliotheque d'apps** : Balayez jusqu'a la derniere page de votre ecran d'accueil pour trouver la **Bibliotheque d'apps**, qui affiche toutes vos applications installees organisees par categorie.
