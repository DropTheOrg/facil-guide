## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a trouver et installer une application sur votre telephone Android. Les applications vous permettent de faire beaucoup de choses comme envoyer des messages, consulter la meteo, jouer a des jeux et bien plus encore.

## Ce dont vous avez besoin

- Un telephone Android
- Une connexion Internet (le Wi-Fi est recommande)
- Un compte Google (votre compte Gmail)

## Les etapes

### Etape 1 : Ouvrir le Google Play Store

Trouvez l'icone du **Play Store** sur votre ecran d'accueil. Elle ressemble a un triangle colore pointant vers la droite. **Appuyez dessus** pour l'ouvrir.

### Etape 2 : Rechercher l'application

En haut de l'ecran, vous verrez une **barre de recherche**. Appuyez dessus et tapez le nom de l'application que vous voulez installer. Par exemple, tapez "WhatsApp" ou "meteo". Puis appuyez sur l'icone de la **loupe** ou sur **Entree** sur votre clavier.

### Etape 3 : Choisir la bonne application

Une liste d'applications apparaitra. Cherchez celle que vous voulez. Verifiez le nom de l'application, le nom du developpeur en dessous et la note en etoiles. Appuyez sur l'application pour voir plus de details.

### Etape 4 : Appuyer sur Installer

Sur la page de l'application, appuyez sur le bouton vert **"Installer"**. Si l'application est payante, le bouton affichera le prix. Pour les applications gratuites, il indiquera simplement **Installer**.

### Etape 5 : Attendre le telechargement

L'application va commencer a se telecharger et s'installer. Vous verrez une barre de progression. Cela prend generalement moins d'une minute en Wi-Fi. Ne fermez pas le Play Store pendant le telechargement.

### Etape 6 : Ouvrir l'application

Une fois installee, le bouton changera en **"Ouvrir"**. Appuyez dessus pour commencer a utiliser l'application. Vous trouverez aussi l'icone de l'application sur votre ecran d'accueil.

## Conseils utiles

- **Mises a jour** : Les applications se mettent a jour regulierement. Pour qu'elles fonctionnent bien, ouvrez le Play Store, appuyez sur votre photo de profil, puis sur **"Gerer les applications et l'appareil"** pour tout mettre a jour.
- **Espace de stockage** : Si votre telephone indique qu'il n'y a pas assez d'espace, vous devrez peut-etre supprimer des photos ou des applications inutilisees avant d'en installer une nouvelle.
- **Utilisez uniquement le Play Store** : Evitez d'installer des applications depuis des sites inconnus. Le Play Store verifie la securite des applications avant de les rendre disponibles.
