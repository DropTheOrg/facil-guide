## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a changer le mot de passe de votre reseau Wi-Fi. C'est important pour proteger votre connexion Internet et empecher des inconnus de l'utiliser.

## Ce dont vous avez besoin

- Un appareil connecte a votre Wi-Fi (ordinateur, telephone ou tablette)
- Un navigateur Internet (Safari, Chrome ou Firefox)
- Les identifiants de connexion de votre box Internet (souvent imprimes sur une etiquette sur la box)

## Les etapes

### Etape 1 : Trouver l'adresse de votre box Internet

Votre box Internet a une adresse speciale que vous tapez dans votre navigateur. Les adresses les plus courantes sont **192.168.1.1** ou **192.168.0.1**. Cette adresse est souvent indiquee sur une etiquette sous ou derriere votre box.

### Etape 2 : Ouvrir la page de configuration

Ouvrez votre navigateur Internet (Safari, Chrome ou Firefox). Dans la barre d'adresse en haut, tapez l'adresse de la box (par exemple **192.168.1.1**) et appuyez sur **Entree** ou **Aller**. Une page de connexion apparaitra.

### Etape 3 : Se connecter a la box

Entrez le **nom d'utilisateur** et le **mot de passe** de votre box. Ce ne sont pas les memes que votre mot de passe Wi-Fi. Les identifiants par defaut sont souvent imprimes sur l'etiquette de la box. Par exemple "admin" pour les deux.

Si vous les avez changes et que vous ne vous en souvenez plus, vous devrez peut-etre reinitialiser votre box en appuyant sur le petit bouton de reinitialisation a l'arriere avec un trombone pendant 10 secondes.

### Etape 4 : Trouver les parametres Wi-Fi

Cherchez une section appelee **"Sans fil"**, **"Wi-Fi"** ou **"Parametres sans fil"**. Elle se trouve generalement dans le menu principal a gauche ou en haut de la page. Appuyez ou cliquez dessus.

### Etape 5 : Changer le mot de passe

Trouvez le champ appele **"Mot de passe"**, **"Cle de securite"** ou **"Cle WPA"**. Supprimez l'ancien mot de passe et tapez le nouveau. Choisissez un mot de passe d'au moins 12 caracteres avec un melange de lettres, chiffres et symboles.

Notez votre nouveau mot de passe sur un papier et gardez-le en lieu sur.

### Etape 6 : Enregistrer les modifications

Cliquez sur le bouton **"Enregistrer"** ou **"Appliquer"**. Votre box peut redemarrer, ce qui prend environ une minute. Une fois qu'elle est revenue, connectez vos appareils avec le nouveau mot de passe.

## Conseils utiles

- **Notez-le** : Gardez votre nouveau mot de passe ecrit sur un papier pres de votre box pour ne pas l'oublier.
- **Reconnectez vos appareils** : Apres le changement, vous devrez entrer le nouveau mot de passe sur chaque appareil (telephones, tablettes, ordinateurs, televiseurs connectes).
- **Type de securite** : Si vous voyez une option pour le type de securite, choisissez **WPA2** ou **WPA3**. Ce sont les options les plus securisees.
