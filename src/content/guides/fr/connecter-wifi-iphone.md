## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a connecter votre iPhone a un reseau Wi-Fi. Utiliser le Wi-Fi vous donne une connexion Internet plus rapide et economise vos donnees mobiles.

## Ce dont vous avez besoin

- Un iPhone (n'importe quel modele)
- Le nom et le mot de passe d'un reseau Wi-Fi

## Les etapes

### Etape 1 : Ouvrir les Reglages

Trouvez l'application **Reglages** sur votre ecran d'accueil. Son icone ressemble a des engrenages gris. Appuyez dessus.

### Etape 2 : Appuyer sur Wi-Fi

En haut de l'ecran des Reglages, vous verrez **Wi-Fi**. Appuyez dessus. Verifiez que le bouton a cote de Wi-Fi est active (vert). S'il est gris, appuyez dessus pour l'activer.

### Etape 3 : Choisir votre reseau

Votre iPhone affichera une liste des reseaux Wi-Fi disponibles. Trouvez le nom du reseau auquel vous voulez vous connecter et appuyez dessus. C'est souvent le nom de votre routeur ou le nom donne par l'endroit ou vous etes (comme un cafe ou un hotel).

### Etape 4 : Entrer le mot de passe

Une boite apparaitra demandant le mot de passe. Tapez le mot de passe Wi-Fi soigneusement avec le clavier. Les mots de passe sont sensibles aux majuscules et minuscules, ce qui signifie que les lettres capitales et petites comptent. Quand vous avez tape le mot de passe, appuyez sur **Rejoindre** en haut a droite.

### Etape 5 : Confirmer la connexion

Si le mot de passe est correct, une coche bleue apparaitra a cote du nom du reseau. Vous verrez aussi l'icone Wi-Fi (des lignes courbes) en haut a droite de votre ecran. Vous etes maintenant connecte.

## Conseils utiles

- **Reconnexion automatique** : Votre iPhone se souvient des reseaux Wi-Fi auxquels vous vous etes deja connecte. Il se connectera automatiquement la prochaine fois que vous serez a proximite.
- **Oublier un reseau** : Si vous voulez empecher votre telephone de se connecter automatiquement a un reseau, appuyez sur l'icone bleue d'information (i) a cote du nom du reseau, puis appuyez sur **Oublier ce reseau**.
- **Verifier votre connexion** : Si Internet semble lent, essayez de desactiver et reactiver le Wi-Fi dans les Reglages. Cela resout souvent les problemes temporaires.
