## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a bloquer un numero de telephone sur votre iPhone. Bloquer un numero empeche cette personne de vous appeler, de vous envoyer des SMS ou de vous joindre sur FaceTime. C'est utile pour stopper les appels indesirables ou le spam.

## Ce dont vous avez besoin

- Un iPhone
- Le numero de telephone que vous voulez bloquer (soit depuis un appel recent, soit depuis vos contacts)

## Les etapes

### Etape 1 : Ouvrir l'application Telephone

Appuyez sur l'icone verte **Telephone** sur votre ecran d'accueil. Elle ressemble a un combine telephonique blanc sur fond vert.

### Etape 2 : Trouver le numero a bloquer

Appuyez sur **"Recents"** en bas de l'ecran pour voir vos appels recents. Trouvez le numero ou le contact que vous voulez bloquer dans la liste.

Si le numero est enregistre comme contact, vous pouvez aussi le trouver dans l'onglet **"Contacts"**.

### Etape 3 : Appuyer sur le bouton d'information

A cote du numero de telephone, appuyez sur la petite icone bleue **"i"** (bouton d'information) sur le cote droit. Les details de cet appelant s'afficheront.

### Etape 4 : Defiler vers le bas et appuyer sur Bloquer ce correspondant

Defilez jusqu'en bas de l'ecran. Vous verrez un texte rouge indiquant **"Bloquer ce correspondant"**. Appuyez dessus.

### Etape 5 : Confirmer le blocage

Un message apparaitra pour vous demander de confirmer. Appuyez sur **"Bloquer le contact"** pour terminer. Le numero est maintenant bloque et ne pourra plus vous appeler ni vous envoyer de messages.

## Conseils utiles

- **Bloquer depuis Messages** : Vous pouvez aussi bloquer un numero depuis l'application Messages. Ouvrez la conversation, appuyez sur le nom en haut, appuyez sur l'icone **info**, puis defilez jusqu'a **"Bloquer ce correspondant"**.
- **Voir votre liste de bloques** : Allez dans **Reglages**, puis **Telephone**, puis **Contacts bloques** pour voir toutes les personnes que vous avez bloquees.
- **Appels spam** : Si vous recevez beaucoup d'appels spam, activez **"Appels d'inconnus silencieux"** dans Reglages > Telephone. Les appels de numeros inconnus iront directement sur la messagerie vocale.
