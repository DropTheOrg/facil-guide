## Ce que vous allez apprendre

Dans ce guide, vous apprendrez à configurer la messagerie vocale sur votre téléphone pour que les personnes qui vous appellent puissent vous laisser un message vocal quand vous ne pouvez pas répondre. Vous apprendrez aussi à écouter vos messages, les sauvegarder ou les supprimer.

## Ce dont vous avez besoin

- Un iPhone ou un téléphone Android avec un forfait mobile actif
- Le code PIN ou le code d'accès de votre téléphone
- Un endroit calme pour enregistrer votre annonce

## Les étapes

### Étape 1 : Ouvrir l'application Téléphone

Appuyez sur l'icône **Téléphone** sur votre écran d'accueil. Elle ressemble à un combiné téléphonique vert sur iPhone, ou à une icône de téléphone bleue ou verte sur Android. C'est la même application que vous utilisez pour passer des appels normaux.

### Étape 2 : Accéder aux paramètres de la messagerie vocale

Sur **iPhone** : appuyez sur l'onglet **Messagerie** en bas à droite de l'application Téléphone. Si c'est la première fois, vous verrez un bouton **Configurer maintenant**. Appuyez dessus.

Sur **Android** : ouvrez l'application **Téléphone**, appuyez sur les trois points en haut à droite, puis sur **Paramètres**. Cherchez **Messagerie vocale** et appuyez dessus. Vous devrez peut-être faire défiler vers le bas pour la trouver.

### Étape 3 : Créer un code PIN de messagerie

Votre téléphone vous demandera de créer un code PIN (un code numérique court, généralement de 4 à 6 chiffres). Ce code protège votre messagerie pour que vous seul puissiez écouter vos messages. Choisissez un numéro dont vous vous souviendrez, tapez-le et confirmez-le en le retapant. Notez ce code PIN et gardez-le dans un endroit sûr.

### Étape 4 : Enregistrer votre annonce personnelle

Après avoir défini le code PIN, on vous demandera d'enregistrer une annonce. C'est le message que les appelants entendent quand ils arrivent sur votre messagerie. Appuyez sur **Personnalisé** ou **Personnel** pour enregistrer votre propre annonce. Parlez clairement et dites quelque chose comme : "Bonjour, vous êtes bien sur le répondeur de [votre nom]. Je ne suis pas disponible pour le moment. Veuillez laisser un message après le bip et je vous rappellerai." Appuyez sur **Enregistrer** ou **Terminé** quand vous avez fini.

### Étape 5 : Écouter un message vocal

Quand quelqu'un vous laisse un message vocal, vous verrez une notification sur votre téléphone. Sur **iPhone**, appuyez sur l'onglet **Messagerie** dans l'application Téléphone. Vous verrez une liste de messages avec le nom ou le numéro de l'appelant et l'heure de l'appel. Appuyez sur un message, puis sur le **bouton lecture** pour l'écouter.

Sur **Android**, vous pouvez généralement appeler votre messagerie en maintenant enfoncé le chiffre **1** sur le clavier du téléphone, ou en appuyant sur la notification de messagerie. Suivez les instructions vocales pour écouter.

### Étape 6 : Supprimer les anciens messages

Pour garder votre messagerie en ordre, supprimez les messages dont vous n'avez plus besoin. Sur **iPhone**, balayez vers la gauche sur un message et appuyez sur **Supprimer**. Sur **Android**, après avoir écouté un message, suivez l'instruction vocale pour le supprimer (généralement en appuyant sur une touche comme le **7**). Vider régulièrement les anciens messages garantit que les nouveaux appelants pourront toujours vous laisser un message.

## Conseils

- Enregistrez votre annonce dans une pièce calme pour que les appelants puissent bien vous entendre.
- Si vous oubliez votre code PIN de messagerie, contactez votre opérateur mobile. Il pourra le réinitialiser.
- Sur iPhone, les messages vocaux sont stockés sur votre téléphone. Si vous changez de téléphone, vous risquez de perdre les messages sauvegardés s'ils ne sont pas sauvegardés.
- Certains opérateurs proposent la messagerie visuelle, qui affiche vos messages sous forme de liste. Demandez à votre opérateur si cette option est disponible avec votre forfait.
