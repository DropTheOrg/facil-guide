## Ce que vous allez apprendre

Dans ce guide, vous apprendrez à accéder à votre compte Ameli et à l'utiliser pour gérer votre assurance maladie. Vous verrez comment vérifier si une consultation a été remboursée, télécharger des documents officiels et mettre à jour vos informations personnelles. Ameli est le service en ligne de l'Assurance Maladie, le système public d'assurance santé en France.

## Ce dont vous avez besoin

- Votre numéro de sécurité sociale
- Un téléphone, une tablette ou un ordinateur avec un accès internet
- Une adresse email associée à votre compte Ameli
- Votre mot de passe Ameli (ou vos identifiants FranceConnect)

## Les étapes

### Étape 1 : Télécharger l'application Ameli ou ouvrir le site

Sur votre téléphone, téléchargez l'application **Ameli** depuis l'**App Store** (iPhone) ou le **Google Play Store** (Android). Vous pouvez aussi ouvrir votre navigateur internet et aller sur [ameli.fr](https://www.ameli.fr).

### Étape 2 : Se connecter à votre compte

Appuyez sur **Mon compte** en haut de l'écran. Entrez votre numéro de sécurité sociale (13 chiffres, présent sur votre carte Vitale ou sur un courrier Ameli) et votre mot de passe. Vous pouvez aussi appuyer sur **FranceConnect** pour vous connecter avec vos identifiants FranceConnect.

### Étape 3 : Vérifier vos remboursements

Une fois connecté, appuyez sur **Mes paiements** dans le menu. Vous verrez la liste de vos dépenses de santé récentes et le montant remboursé par l'Assurance Maladie. Chaque ligne affiche la date, le professionnel de santé, le montant facturé et le montant remboursé.

### Étape 4 : Télécharger une attestation

Pour obtenir un justificatif d'assurance, appuyez sur **Mes démarches** puis sur **Obtenir une attestation**. Choisissez le type d'attestation dont vous avez besoin, comme **Attestation de droits**. Appuyez sur **Télécharger** et le document sera enregistré en PDF sur votre appareil.

### Étape 5 : Mettre à jour vos informations personnelles

Si vous avez déménagé ou changé de compte bancaire, appuyez sur **Mon profil**. Ici vous pouvez modifier votre adresse, numéro de téléphone, email et coordonnées bancaires pour recevoir les remboursements. Les modifications prennent généralement effet en quelques jours.

### Étape 6 : Commander une nouvelle carte Vitale

Si votre carte Vitale est perdue, volée ou abîmée, appuyez sur **Mes démarches** puis **Commander une carte Vitale**. Suivez les instructions pour télécharger une photo d'identité et confirmer votre adresse. La nouvelle carte vous sera envoyée par courrier en quelques semaines.

### Étape 7 : Contacter votre caisse d'assurance maladie

Si vous avez une question que vous ne pouvez pas résoudre en ligne, appuyez sur **Mes messages** pour envoyer un message sécurisé à votre CPAM. Vous pouvez aussi trouver l'adresse et le numéro de téléphone de votre caisse sous **Mon organisme**.

## Conseils

- Vérifiez vos remboursements environ 5 jours après une consultation. Il faut généralement 3 à 7 jours pour que le remboursement apparaisse.
- Gardez votre adresse email à jour dans Ameli pour recevoir les notifications de nouveaux remboursements et les courriers importants.
- Si vous avez une mutuelle, le reste à charge après le remboursement Ameli est généralement payé automatiquement par votre mutuelle si elle est télétransmise.
- Vous pouvez activer les notifications dans l'application Ameli pour être prévenu à chaque nouveau remboursement.
