## Ce que vous allez apprendre

Vous allez apprendre a transformer votre telephone en point d'acces Wi-Fi pour que d'autres appareils comme un ordinateur portable, une tablette ou un autre telephone puissent utiliser votre connexion internet mobile. C'est utile quand il n'y a pas de Wi-Fi disponible.

## Ce dont vous avez besoin

- Un telephone (iPhone ou Android) avec des donnees mobiles
- Un forfait mobile qui autorise le partage de connexion
- L'appareil que vous souhaitez connecter (ordinateur, tablette ou autre telephone)

### Etape 1 : Ouvrir les reglages du partage de connexion (iPhone)

Ouvrez l'application **Reglages**. Appuyez sur **Partage de connexion**. Si vous ne voyez pas cette option, appuyez d'abord sur **Donnees mobiles**, puis sur **Partage de connexion**. Vous verrez un bouton pour activer ou desactiver le partage et un mot de passe Wi-Fi.

### Etape 2 : Ouvrir les reglages du partage de connexion (Android)

Ouvrez l'application **Parametres**. Appuyez sur **Connexions** ou **Reseau et Internet** (le nom varie selon le telephone). Appuyez sur **Point d'acces mobile et modem** ou **Partage de connexion**. Appuyez sur **Point d'acces mobile** pour voir les parametres.

### Etape 3 : Definir un mot de passe

Avant d'activer le partage, verifiez le mot de passe. Sur iPhone, appuyez sur le champ **Mot de passe Wi-Fi** pour le modifier. Sur Android, appuyez sur **Configurer** ou le nom du point d'acces pour voir et modifier le mot de passe. Choisissez un mot de passe facile a retenir pour vous mais difficile a deviner pour les autres. Utilisez au moins 8 caracteres.

### Etape 4 : Activer le partage de connexion

Activez le bouton a cote de **Autoriser d'autres utilisateurs** (iPhone) ou **Point d'acces mobile** (Android). Votre telephone diffuse maintenant un reseau Wi-Fi auquel d'autres appareils peuvent se connecter. Le nom du reseau est generalement le nom de votre telephone.

### Etape 5 : Connecter votre autre appareil

Sur l'appareil que vous souhaitez connecter (ordinateur, tablette ou autre telephone), ouvrez les parametres Wi-Fi. Cherchez le nom de votre telephone dans la liste des reseaux disponibles. Appuyez dessus et entrez le mot de passe defini a l'etape 3. L'appareil devrait se connecter en quelques secondes.

### Etape 6 : Desactiver le partage quand vous avez termine

Quand vous n'avez plus besoin du partage, retournez dans les parametres et desactivez le bouton. Cela economise la batterie et empeche les autres d'utiliser vos donnees mobiles. Sur iPhone, le partage se desactive automatiquement quand aucun appareil n'est connecte pendant quelques minutes.

## Conseils

- Gardez votre telephone en charge pendant l'utilisation du partage car il vide la batterie rapidement
- Utilisez un mot de passe fort pour empecher des inconnus de se connecter
- Verifiez votre consommation de donnees dans les Reglages apres avoir utilise le partage
- Si la connexion est lente, essayez de vous deplacer vers un endroit avec un meilleur signal mobile
- Certains forfaits facturent un supplement pour le partage de connexion -- verifiez aupres de votre operateur
