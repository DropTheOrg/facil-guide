## Ce que vous allez apprendre

Dans ce guide, vous apprendrez a verifier si une mise a jour est disponible pour votre iPhone et a l'installer. Garder votre iPhone a jour vous donne les dernieres fonctionnalites, corrige les problemes et vous protege contre les menaces de securite.

## Ce dont vous avez besoin

- Un iPhone
- Une connexion Wi-Fi (les mises a jour sont volumineuses et ne peuvent pas etre telechargees en donnees mobiles)
- Au moins 50% de batterie ou votre chargeur branche
- Suffisamment d'espace de stockage libre (generalement 2-5 Go)

## Les etapes

### Etape 1 : Se connecter au Wi-Fi

Assurez-vous que votre iPhone est connecte a un reseau Wi-Fi. Les mises a jour sont trop volumineuses pour etre telechargees en donnees mobiles. Si vous n'etes pas connecte, allez dans **Reglages** puis **Wi-Fi** et appuyez sur le nom de votre reseau.

### Etape 2 : Ouvrir les Reglages

Appuyez sur l'application **Reglages** sur votre ecran d'accueil. Elle a une icone grise en forme d'engrenages.

### Etape 3 : Aller dans Mise a jour logicielle

Faites defiler vers le bas et appuyez sur **"General"**. Puis appuyez sur **"Mise a jour logicielle"**. Votre iPhone verifiera si une nouvelle mise a jour est disponible. Cela peut prendre quelques secondes.

### Etape 4 : Lire les informations sur la mise a jour

Si une mise a jour est disponible, vous verrez son nom et une description des nouveautes. Lisez-la pour comprendre les changements. Si vous voyez "Votre logiciel est a jour", votre iPhone a deja la derniere version.

### Etape 5 : Telecharger et installer

Appuyez sur **"Telecharger et installer"**. Votre iPhone peut vous demander de saisir votre code d'acces (le numero que vous utilisez pour deverrouiller votre telephone). Saisissez-le. Le telechargement commencera. Vous pouvez voir la barre de progression a l'ecran.

### Etape 6 : Attendre la fin de la mise a jour

Une fois le telechargement termine, votre iPhone redemarrera et installera la mise a jour. Vous verrez le logo Apple et une barre de progression. N'eteignez pas votre telephone pendant ce processus. Quand la mise a jour est terminee, votre telephone se rallumera et vous demandera de saisir votre code d'acces.

## Conseils utiles

- **Charger votre telephone** : Branchez votre chargeur avant de commencer la mise a jour. Si votre batterie est en dessous de 50%, la mise a jour ne demarrera pas.
- **Mettre a jour la nuit** : Vous pouvez programmer les mises a jour pour la nuit. Quand l'ecran de mise a jour apparait, appuyez sur **"Installer cette nuit"**.
- **Mises a jour automatiques** : Allez dans **Reglages** puis **General** puis **Mise a jour logicielle** puis **Mises a jour automatiques** pour que votre iPhone se mette a jour automatiquement.
- **Sauvegarder d'abord** : Avant toute mise a jour majeure, sauvegardez votre iPhone sur iCloud. Allez dans **Reglages**, appuyez sur votre nom en haut, puis **iCloud** puis **Sauvegarde iCloud**.
