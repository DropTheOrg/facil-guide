## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a desactiver les notifications pour des applications specifiques sur votre iPhone. C'est utile quand une application envoie trop d'alertes et que vous voulez un telephone plus calme.

## Ce dont vous avez besoin

- Un iPhone (n'importe quel modele)

## Les etapes

### Etape 1 : Ouvrir les Reglages

Trouvez l'application **Reglages** sur votre ecran d'accueil. Son icone ressemble a des engrenages gris. Appuyez dessus pour l'ouvrir.

### Etape 2 : Appuyer sur Notifications

Faites defiler legerement vers le bas et appuyez sur **Notifications**. Cela ouvre une liste de toutes les applications installees sur votre iPhone.

### Etape 3 : Choisir l'application a rendre silencieuse

Faites defiler la liste et trouvez l'application dont vous voulez desactiver les notifications. Par exemple, si vous voulez arreter les alertes d'un jeu, trouvez le nom du jeu dans la liste et appuyez dessus.

### Etape 4 : Desactiver Autoriser les notifications

En haut de l'ecran, vous verrez un bouton a cote de **Autoriser les notifications**. Appuyez sur le bouton pour le desactiver. Quand le bouton est gris (pas vert), les notifications de cette application sont desactivees.

### Etape 5 : Repeter pour d'autres applications si necessaire

Appuyez sur la fleche retour en haut a gauche pour revenir a la liste complete. Vous pouvez repeter les etapes 3 et 4 pour toute autre application que vous souhaitez rendre silencieuse.

## Conseils utiles

- **Methode rapide depuis l'ecran verrouille** : Quand vous voyez une notification sur votre ecran verrouille, balayez-la vers la gauche. Appuyez sur **Options**, puis sur **Desactiver** pour desactiver instantanement les notifications de cette application.
- **Resume programme** : Au lieu de desactiver completement les notifications, vous pouvez activer **Resume programme** dans les reglages de notification d'une application. Cela regroupe les alertes et les affiche a l'heure que vous choisissez.
- **Mode Concentration** : Si vous voulez du calme sans desactiver les notifications definitivement, essayez le mode **Concentration** depuis le Centre de controle.
