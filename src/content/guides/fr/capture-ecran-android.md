## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a faire une capture d'ecran sur votre telephone Android. Une capture d'ecran est une photo de ce qui est affiche sur votre ecran. C'est utile pour sauvegarder une information, partager ce que vous voyez ou garder une trace.

## Ce dont vous avez besoin

- Un telephone Android (toute marque : Samsung, Google Pixel, Xiaomi, Huawei ou autre)

## Les etapes

### Etape 1 : Allez sur ce que vous voulez capturer

Ouvrez l'application, le message ou l'ecran que vous souhaitez sauvegarder en capture d'ecran. Tout ce qui est visible sur votre ecran sera capture.

### Etape 2 : Trouvez les bons boutons

Sur la plupart des telephones Android, vous utiliserez deux boutons :

- Le **bouton Marche/Arret** (generalement sur le cote droit du telephone)
- Le **bouton Volume Bas** (generalement sur le cote gauche ou sous le bouton Marche/Arret)

### Etape 3 : Appuyez sur les deux boutons en meme temps

Appuyez sur le **bouton Marche/Arret** et le **bouton Volume Bas** en meme temps. Maintenez-les environ une seconde, puis relachez les deux.

Votre ecran va clignoter brievement et vous entendrez peut-etre un son d'obturateur. Cela signifie que la capture d'ecran a ete prise avec succes.

### Etape 4 : Trouvez l'apercu ou la notification

Apres la capture, un petit apercu apparaitra en bas ou en haut de votre ecran. Vous verrez peut-etre aussi une notification. Appuyez sur l'apercu pour modifier ou partager la capture immediatement.

### Etape 5 : Retrouvez votre capture dans la Galerie

Ouvrez votre application **Galerie** ou **Google Photos**. Cherchez un dossier ou album appele **"Captures d'ecran"**. Votre nouvelle capture s'y trouvera.

### Etape 6 : Methode alternative avec les Parametres rapides

Si la methode avec les boutons est difficile, essayez ceci :

1. Faites glisser vers le bas depuis le **haut de votre ecran** avec deux doigts pour ouvrir les **Parametres rapides**
2. Cherchez un bouton intitule **"Capture d'ecran"** ou une petite icone de ciseaux
3. Appuyez dessus pour capturer l'ecran

Tous les telephones n'affichent pas cette option par defaut, mais beaucoup de telephones Samsung et Google Pixel l'incluent.

## Conseils utiles

- **Partager rapidement** : Apres avoir fait une capture, appuyez sur le bouton **Partager** dans la notification d'apercu pour l'envoyer par WhatsApp, email ou message.
- **Telephones Samsung** : Sur certains Samsung, vous pouvez aussi faire glisser le tranchant de votre main sur l'ecran pour capturer. Activez cette option dans **Parametres** puis **Fonctions avancees** puis **Mouvements et gestes**.
- **Google Assistant** : Dites "Hey Google, fais une capture d'ecran" pour capturer l'ecran sans les mains.
- **Capture d'ecran longue** : Sur beaucoup de telephones Android modernes, apres une capture vous verrez un bouton **"Capturer plus"**. Appuyez dessus pour capturer une page longue qui depasse ce qui est visible.
