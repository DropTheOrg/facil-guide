## Ce que vous allez apprendre

Dans ce guide, vous apprendrez comment copier du texte d'un endroit sur votre telephone et le coller ailleurs. C'est utile pour partager des adresses, des numeros de telephone, des liens ou tout texte sans avoir a le retaper.

## Ce dont vous avez besoin

- Votre telephone (iPhone ou Android)

## Les etapes

### Etape 1 : Trouver le texte que vous voulez copier

Ouvrez l'application, le message, le site web ou le document qui contient le texte que vous voulez copier. Il peut s'agir d'une adresse web, d'un numero de telephone, d'un paragraphe ou meme d'un seul mot.

### Etape 2 : Appuyer longuement sur le texte

Placez votre doigt sur le texte que vous voulez copier et **appuyez longuement** pendant environ une seconde. Ne tapez pas rapidement -- maintenez votre doigt appuye. Vous verrez le texte se surligner en bleu avec de petites poignees qui apparaissent autour.

### Etape 3 : Ajuster la selection

Une fois le texte surligne, vous verrez deux petites **poignees** (cercles ou lignes) au debut et a la fin de la selection. **Faites glisser ces poignees** pour selectionner exactement le texte que vous voulez. Faites glisser la poignee gauche vers le debut souhaite et la droite vers la fin.

### Etape 4 : Appuyer sur Copier

Une fois que vous avez selectionne le texte souhaite, un petit menu apparaitra au-dessus ou en dessous de la selection. Appuyez sur **Copier**. Le texte est maintenant sauvegarde dans votre presse-papiers (une zone de stockage temporaire sur votre telephone). Rien de visible ne se passera, mais le texte est copie.

### Etape 5 : Coller le texte ou vous le souhaitez

Allez a l'endroit ou vous voulez mettre le texte -- un message, un e-mail, une note ou une barre de recherche. Appuyez sur le champ de texte pour y placer votre curseur. Puis **appuyez longuement** sur le champ vide pendant environ une seconde. Un menu apparaitra. Appuyez sur **Coller**. Le texte que vous avez copie apparaitra.

## Conseils

- Pour selectionner tout le texte d'un champ rapidement, appuyez sur **Tout selectionner** dans le menu qui apparait apres un appui long.
- Sur iPhone, vous pouvez aussi secouer votre telephone pour annuler un collage si vous faites une erreur.
- Sur les telephones Samsung, le clavier a souvent un bouton **presse-papiers** qui montre tout ce que vous avez copie recemment.
- Entrainez-vous d'abord avec un simple message texte jusqu'a ce que vous soyez a l'aise.
