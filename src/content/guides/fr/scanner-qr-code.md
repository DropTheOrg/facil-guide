## Ce dont vous aurez besoin

- Votre iPhone ou téléphone Android
- Un QR code à scanner (sur du papier, un écran ou une affiche)

## Étape 1 : Ouvrez votre appareil photo

Prenez votre téléphone et ouvrez l'application **Appareil photo**. C'est la même application que vous utilisez pour prendre des photos.

- **Sur iPhone :** Touchez l'icône Appareil photo sur votre écran d'accueil
- **Sur Android :** Touchez l'icône Appareil photo sur votre écran d'accueil ou glissez depuis l'écran de verrouillage

Vous n'avez pas besoin de prendre une photo. Ouvrez simplement l'appareil photo.

## Étape 2 : Pointez votre appareil photo vers le QR code

Tenez votre téléphone pour que l'appareil photo puisse voir le QR code. Gardez le téléphone à environ **15 à 25 centimètres** du code.

Assurez-vous que :
- Le QR code entier est visible sur votre écran
- Le code est bien éclairé (pas dans l'ombre)
- Votre main est stable

## Étape 3 : Attendez que votre téléphone le reconnaisse

Votre téléphone va détecter automatiquement le QR code. Vous n'avez pas besoin d'appuyer sur un bouton.

- **Sur iPhone :** Une notification jaune apparaîtra en haut de l'écran avec un lien
- **Sur Android :** Une petite bannière ou bulle apparaîtra avec le lien

Cela prend généralement 1 à 3 secondes. Gardez le téléphone immobile.

## Étape 4 : Touchez le lien

Quand la notification ou la bannière apparaît :

- **Sur iPhone :** Touchez la notification jaune en haut de l'écran
- **Sur Android :** Touchez le lien ou la bannière qui est apparue

Cela ouvrira le site web ou le contenu que le QR code contient. Il s'ouvre généralement dans votre navigateur web (Safari ou Chrome).

## Étape 5 : Si cela ne fonctionne pas

Si votre appareil photo ne reconnaît pas le QR code :

1. **Vérifiez l'éclairage.** Déplacez-vous vers un endroit plus lumineux ou allumez une lumière
2. **Nettoyez votre appareil photo.** Essuyez doucement l'objectif avec un tissu doux
3. **Essayez une distance différente.** Rapprochez ou éloignez votre téléphone du code
4. **Vérifiez vos réglages :**
   - Sur iPhone : Allez dans **Réglages** > **Appareil photo** > Assurez-vous que **Scanner les codes QR** est activé
   - Sur Android : Ouvrez **Appareil photo** > Touchez **Réglages** (icône roue dentée) > Cherchez **Scanner les codes QR** et activez-le

Si rien ne fonctionne, demandez de l'aide à quelqu'un. Certains téléphones plus anciens peuvent nécessiter une application gratuite de lecture de QR codes depuis l'App Store ou Google Play.

## C'est fait !

Vous savez maintenant scanner n'importe quel QR code. Vous les trouverez partout -- menus de restaurants, arrêts de bus, billets d'événements, emballages de produits, et même dans des guides comme celui-ci.
