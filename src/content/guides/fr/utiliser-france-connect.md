## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a utiliser **France Connect** pour acceder aux services publics en ligne comme les impots, l'Assurance Maladie (Ameli), la retraite, et bien d'autres. France Connect vous evite de retenir un mot de passe different pour chaque site du gouvernement.

## Ce dont vous avez besoin

- Un ordinateur, une tablette ou un telephone avec Internet
- Un compte existant sur l'un de ces services : **Ameli** (Assurance Maladie), **impots.gouv.fr**, ou **L'Identite Numerique La Poste**
- Le mot de passe de ce compte

## Les etapes

### Etape 1 : Comprendre ce qu'est France Connect

France Connect est un **bouton** que vous trouvez sur les sites du gouvernement. Au lieu de creer un nouveau compte sur chaque site, vous utilisez un compte que vous avez deja (comme Ameli ou les impots) pour vous connecter partout.

C'est comme une cle universelle : une seule cle ouvre toutes les portes des services publics.

### Etape 2 : Aller sur le site du service public

Ouvrez votre navigateur Internet et allez sur le site du service auquel vous voulez acceder. Par exemple :
- **impots.gouv.fr** pour les impots
- **ameli.fr** pour l'Assurance Maladie
- **info-retraite.fr** pour la retraite
- **france-services.gouv.fr** pour d'autres services

### Etape 3 : Trouver le bouton France Connect

Sur la page de connexion du site, cherchez le bouton **France Connect**. C'est un bouton bleu avec le logo France Connect (un hexagone tricolore). Il est generalement place au-dessus ou a cote des champs de connexion habituels.

Cliquez ou appuyez sur ce bouton.

### Etape 4 : Choisir votre compte de connexion

Une fenetre s'ouvre avec une liste de comptes que vous pouvez utiliser :
- **Ameli** (le plus courant)
- **impots.gouv.fr**
- **L'Identite Numerique La Poste**
- **MobileConnect et moi**
- **Yris** (identite numerique)

Choisissez le compte dont vous connaissez le mot de passe. **Ameli** est souvent le plus facile car beaucoup de personnes ont deja un compte.

### Etape 5 : Entrer vos identifiants

Vous etes redirige vers le site du compte que vous avez choisi. Par exemple, si vous avez choisi Ameli :
- Entrez votre **numero de securite sociale** (13 chiffres, sur votre carte Vitale)
- Entrez votre **mot de passe Ameli**
- Cliquez sur **Se connecter**

### Etape 6 : Autoriser le partage d'informations

France Connect vous demande d'autoriser le partage de certaines informations (votre nom, prenom, date de naissance) avec le site que vous voulez utiliser. Ces informations sont necessaires pour vous identifier.

Cliquez sur **Autoriser** ou **Continuer**.

### Etape 7 : Vous etes connecte

Vous etes maintenant connecte au service public. Vous pouvez consulter vos informations, faire vos demarches, ou telecharger vos documents.

Quand vous avez fini, pensez a cliquer sur **Se deconnecter** en haut de la page.

## Conseils utiles

- **Toujours le meme compte** : Utilisez toujours le meme compte (par exemple Ameli) pour vous connecter via France Connect. Cela evite les confusions.
- **Gardez votre carte Vitale a portee de main** : Si vous utilisez Ameli, votre numero de securite sociale est sur votre carte Vitale. Gardez-la pres de vous quand vous vous connectez.
- **Ne partagez jamais votre mot de passe** : France Connect ne vous demandera jamais votre mot de passe par email ou par telephone. Si quelqu'un le demande, c'est une arnaque.
- **Deconnexion** : Si vous utilisez un ordinateur partage (bibliotheque, maison de retraite), deconnectez-vous toujours en cliquant sur **Se deconnecter** puis en fermant le navigateur.
