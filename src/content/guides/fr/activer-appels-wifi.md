## Ce que vous allez apprendre

Dans ce guide, vous apprendrez a activer les appels Wi-Fi sur votre iPhone ou telephone Android. Les appels Wi-Fi vous permettent de passer et recevoir des appels telephoniques via un reseau Wi-Fi au lieu d'utiliser votre signal mobile. C'est tres utile quand vous etes dans un batiment ou le signal mobile est faible mais ou vous avez une bonne connexion Wi-Fi.

## Ce dont vous avez besoin

- Un iPhone ou telephone Android qui prend en charge les appels Wi-Fi (la plupart des telephones fabriques apres 2018)
- Un reseau Wi-Fi auquel vous etes connecte
- Un operateur mobile qui prend en charge les appels Wi-Fi (la plupart des grands operateurs le font)

### Etape 1 : Verifiez si votre operateur prend en charge les appels Wi-Fi

La plupart des grands operateurs mobiles prennent en charge les appels Wi-Fi, mais il est bon de verifier d'abord. Vous pouvez appeler le service client de votre operateur ou visiter son site internet. Si vous n'etes pas sur, essayez simplement de l'activer dans les etapes suivantes. Si l'option n'apparait pas, votre operateur ou votre telephone ne le prend peut-etre pas en charge.

### Etape 2 : Connectez-vous a un reseau Wi-Fi

Assurez-vous que votre telephone est connecte a un reseau Wi-Fi. Ouvrez **Reglages** sur votre telephone et appuyez sur **Wi-Fi**. Choisissez votre reseau dans la liste et entrez le mot de passe si necessaire. Vous devriez voir le symbole Wi-Fi en haut de votre ecran une fois connecte.

### Etape 3 : Activez les appels Wi-Fi sur iPhone

Si vous avez un iPhone :

1. Ouvrez **Reglages**
2. Appuyez sur **Telephone**
3. Appuyez sur **Appels Wi-Fi**
4. Activez l'interrupteur a cote de **Appels Wi-Fi sur cet iPhone**
5. On peut vous demander de confirmer votre adresse d'urgence. Entrez votre adresse et appuyez sur **Enregistrer**

Quand les appels Wi-Fi sont actifs, vous verrez le mot **Wi-Fi** a cote du nom de votre operateur dans la barre d'etat en haut de votre ecran.

### Etape 4 : Activez les appels Wi-Fi sur Android

Si vous avez un telephone Android :

1. Ouvrez **Reglages**
2. Appuyez sur **Reseau et internet** ou **Connexions** (le nom varie selon la marque)
3. Appuyez sur **Appels Wi-Fi** ou cherchez-le sous **Reseau mobile**
4. Activez l'interrupteur pour activer les appels Wi-Fi
5. On peut vous demander d'accepter des conditions ou de confirmer votre adresse d'urgence

Sur les telephones Samsung, allez dans **Reglages**, puis **Connexions**, puis **Appels Wi-Fi**. Sur les telephones Google Pixel, allez dans **Reglages**, puis **Reseau et internet**, puis **Appels et SMS**, puis **Appels Wi-Fi**.

### Etape 5 : Testez les appels Wi-Fi

Pour verifier que cela fonctionne, activez d'abord le mode avion (cela desactive votre signal mobile), puis reactivez le Wi-Fi. Essayez maintenant de passer un appel. Si l'appel aboutit, les appels Wi-Fi fonctionnent correctement. Apres le test, desactivez le mode avion.

### Etape 6 : Definissez votre preference d'appel Wi-Fi

Vous pouvez choisir si votre telephone doit preferer le Wi-Fi ou votre reseau mobile pour les appels.

- **Sur iPhone :** Allez dans **Reglages**, puis **Telephone**, puis **Appels Wi-Fi**. Si vous voyez une option **Preference d'appels Wi-Fi**, vous pouvez choisir entre Wi-Fi prefere ou mobile prefere.
- **Sur Android :** Allez dans les parametres d'appels Wi-Fi (meme chemin qu'a l'etape 4). Cherchez une option de preference. Choisissez **Wi-Fi prefere** si votre signal mobile est generalement faible chez vous, ou **Mobile prefere** si vous ne voulez utiliser les appels Wi-Fi qu'en secours.

## Conseils

- Les appels Wi-Fi sont particulierement utiles dans les sous-sols, les batiments aux murs epais ou les zones rurales ou le signal mobile est faible.
- Si votre Wi-Fi est de mauvaise qualite (lent ou instable), la qualite de vos appels peut aussi etre mauvaise. Une connexion Wi-Fi stable fonctionne le mieux.
- Vous n'avez rien de special a faire pour recevoir des appels en Wi-Fi. Une fois active, les appels entrants utiliseront le Wi-Fi automatiquement quand votre signal mobile est faible.
- Si vous entendez de l'echo ou du retard pendant les appels Wi-Fi, essayez de vous rapprocher de votre routeur Wi-Fi.
