## Ce que vous allez apprendre

Dans ce guide, vous apprendrez a mettre une alarme sur votre iPhone avec l'application Horloge. C'est utile pour vous reveiller le matin, vous rappeler de prendre un medicament ou tout autre rappel.

## Ce dont vous avez besoin

- Un iPhone
- L'application Horloge (elle est preinstallee sur chaque iPhone)

## Les etapes

### Etape 1 : Ouvrir l'application Horloge

Trouvez l'application **Horloge** sur votre ecran d'accueil. Elle a une icone blanche avec un cadran d'horloge. Appuyez dessus. Si vous ne la trouvez pas, balayez vers le bas sur l'ecran d'accueil et tapez "Horloge" dans la barre de recherche.

### Etape 2 : Aller dans l'onglet Alarme

En bas de l'ecran, vous verrez plusieurs onglets : **Horloge mondiale**, **Alarme**, **Chronometre** et **Minuteur**. Appuyez sur **"Alarme"** pour voir la liste de vos alarmes.

### Etape 3 : Creer une nouvelle alarme

Appuyez sur le bouton **"+"** en haut a droite de l'ecran. L'ecran de creation d'alarme s'ouvre pour que vous puissiez regler l'heure et les options.

### Etape 4 : Regler l'heure

Vous verrez une roue de selection qui vous permet de choisir les heures et les minutes. Faites defiler vers le haut ou vers le bas pour selectionner l'heure a laquelle vous voulez que l'alarme sonne. Verifiez bien que vous choisissez **AM** (matin) ou **PM** (apres-midi/soir) correctement.

### Etape 5 : Choisir les jours de repetition et la sonnerie

Appuyez sur **"Repetition"** si vous voulez que cette alarme sonne les memes jours chaque semaine (par exemple, tous les jours de la semaine). Appuyez sur **"Sonnerie"** pour choisir le son de l'alarme. Vous pouvez aussi appuyer sur **"Description"** pour donner un nom a votre alarme comme "Reveil" ou "Medicament".

### Etape 6 : Enregistrer l'alarme

Appuyez sur **"Enregistrer"** en haut a droite. Votre alarme apparait maintenant dans la liste avec un interrupteur vert, ce qui signifie qu'elle est active. Quand l'alarme sonne, balayez l'ecran ou appuyez sur **"Arreter"**.

## Conseils utiles

- **Rappel** : Quand votre alarme sonne, appuyez sur **"Rappel"** pour etre rappele dans 9 minutes. Pour desactiver le rappel, desactivez l'option **"Rappel"** lors de la creation de l'alarme.
- **Volume** : Assurez-vous que le volume de votre telephone est suffisant. Appuyez sur le bouton de volume sur le cote de votre iPhone.
- **Supprimer une alarme** : Balayez vers la gauche sur une alarme dans la liste et appuyez sur **"Supprimer"**.
- **Utiliser Siri** : Vous pouvez aussi dire "Dis Siri, mets une alarme a 7 heures" pour creer une alarme sans ouvrir l'application.
