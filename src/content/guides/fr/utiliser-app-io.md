## Ce que vous allez apprendre

Dans ce guide, vous apprendrez à configurer l'application IO sur votre téléphone et à l'utiliser pour accéder aux services publics italiens. IO vous permet de recevoir des messages des administrations publiques, de payer des impôts et des frais, d'accéder à la Carta Giovani Nazionale et de gérer les bonus gouvernementaux. C'est l'application officielle unique pour interagir numériquement avec les services publics italiens.

## Ce dont vous avez besoin

- Un iPhone ou un téléphone Android
- Une identité numérique SPID ou une CIE (Carta di Identità Elettronica) avec son code PIN
- Une connexion internet

## Les étapes

### Étape 1 : Télécharger l'application IO

Ouvrez l'**App Store** (iPhone) ou le **Google Play Store** (Android) et recherchez **IO**. L'icône de l'application est un cercle blanc sur fond bleu. Appuyez sur **Installer** ou **Obtenir** pour la télécharger.

### Étape 2 : Ouvrir l'application et commencer la connexion

Ouvrez l'application **IO**. Sur l'écran d'accueil, appuyez sur **Se connecter avec SPID** ou **Se connecter avec CIE**. Si vous choisissez SPID, sélectionnez votre fournisseur SPID dans la liste (par exemple **PosteID**, **Aruba** ou **Infocert**). Si vous choisissez CIE, vous aurez besoin de votre carte CIE et de son code PIN.

### Étape 3 : Compléter la connexion

Si vous avez choisi SPID, entrez votre nom d'utilisateur et mot de passe SPID sur la page de connexion du fournisseur. Approuvez la connexion sur l'application de votre fournisseur SPID si demandé. Si vous avez choisi CIE, entrez les 4 premiers chiffres de votre code PIN CIE, puis placez votre carte CIE contre le dos de votre téléphone pour que le lecteur NFC la scanne, puis entrez les 4 chiffres restants.

### Étape 4 : Configurer votre code de sécurité

Après votre première connexion, IO vous demandera de créer un **code de déverrouillage à 6 chiffres**. Choisissez un code que vous pouvez retenir. Vous pouvez aussi activer **Face ID** ou l'**empreinte digitale** pour déverrouiller l'application plus facilement.

### Étape 5 : Explorer vos messages

Appuyez sur l'onglet **Messages** en bas de l'écran. Ici vous verrez les communications officielles des administrations, comme les rappels de paiement, les confirmations de rendez-vous ou les notifications de bonus. Appuyez sur un message pour lire ses détails.

### Étape 6 : Effectuer un paiement

Quand vous recevez un avis de paiement dans vos messages (par exemple un impôt ou une amende), appuyez sur le message puis sur **Payer**. L'application utilise le système **pagoPA**. Choisissez votre moyen de paiement : carte de crédit, carte de débit, PayPal ou virement bancaire. Suivez les instructions pour compléter le paiement. Vous recevrez un reçu dans l'application.

### Étape 7 : Accéder à vos services et bonus

Appuyez sur l'onglet **Services** pour voir quels services gouvernementaux sont disponibles pour vous. Vous pouvez activer ou désactiver les notifications pour chaque service. Si vous avez droit à des bonus (comme le Bonus Cultura ou la Carta Giovani), ils apparaîtront dans l'onglet **Portefeuille** où vous pourrez les activer et les utiliser.

## Conseils

- Gardez les notifications push activées pour l'application IO afin de ne pas manquer les messages importants du gouvernement.
- Vous pouvez enregistrer une carte de paiement dans le portefeuille de l'application une seule fois et l'utiliser pour tous les futurs paiements sans avoir à ressaisir les détails.
- Si la connexion SPID échoue, assurez-vous que l'application de votre fournisseur SPID est à jour et que vous avez approuvé la notification de connexion.
- Consultez régulièrement l'onglet Services car de nouveaux services locaux et nationaux sont ajoutés fréquemment.
