## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a activer le mode sombre sur votre iPhone. Le mode sombre change le fond de l'ecran du blanc au noir, ce qui est plus agreable pour vos yeux dans l'obscurite et peut economiser de la batterie.

## Ce dont vous avez besoin

- Un iPhone sous iOS 13 ou plus recent (la plupart des iPhone a partir de 2015)

## Les etapes

### Etape 1 : Ouvrir les Reglages

Trouvez l'application **Reglages** sur votre ecran d'accueil. Son icone ressemble a des engrenages gris. Appuyez dessus.

### Etape 2 : Appuyer sur Luminosite et affichage

Faites defiler legerement vers le bas et appuyez sur **Luminosite et affichage**. C'est ici que vous controlez l'apparence de votre ecran.

### Etape 3 : Choisir Sombre

En haut de l'ecran, vous verrez deux options : **Clair** et **Sombre**. Chacune affiche un petit apercu de l'apparence de votre telephone. Appuyez sur **Sombre** pour passer en mode sombre.

### Etape 4 : Voir le changement

Votre ecran passera immediatement a un fond sombre avec du texte clair. Vous pouvez maintenant revenir a votre ecran d'accueil et ouvrir des applications pour voir le mode sombre en action.

### Etape 5 : Activer le mode automatique (optionnel)

Si vous voulez que le mode sombre ne s'active que la nuit, activez le bouton **Automatique** sous les options Clair et Sombre. Appuyez sur **Options** pour choisir entre **Du coucher au lever du soleil** ou un horaire personnalise.

## Conseils utiles

- **Basculer rapidement depuis le Centre de controle** : Balayez vers le bas depuis le coin superieur droit de votre ecran, puis appuyez longuement sur le **curseur de luminosite**. Un bouton pour activer ou desactiver le mode sombre apparaitra en bas.
- **Reglages par application** : Certaines applications comme WhatsApp, YouTube et Google Maps ont leur propre reglage de mode sombre. Verifiez dans les reglages de l'application si elle ne suit pas le reglage du systeme.
- **Reduire le point blanc** : Si le mode sombre vous semble encore trop lumineux, allez dans Reglages, Accessibilite, Affichage et taille du texte, et activez **Reduire le point blanc** pour attenuer davantage les couleurs vives.
