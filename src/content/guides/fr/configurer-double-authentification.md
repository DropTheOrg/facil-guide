## Ce que vous allez apprendre

Dans ce guide, vous apprendrez a configurer la double authentification sur vos comptes. Cela ajoute une couche de securite supplementaire pour que personne ne puisse acceder a vos comptes meme en connaissant votre mot de passe. Vous recevrez un code sur votre telephone a chaque connexion depuis un nouvel appareil.

## Ce dont vous avez besoin

- Un telephone qui peut recevoir des SMS
- Acces a vos comptes email, reseaux sociaux ou bancaires
- Quelques minutes pour chaque compte que vous souhaitez proteger

## Etapes

### Etape 1 : Comprendre pourquoi c'est important

Chaque jour, des pirates tentent de voler des mots de passe. Si quelqu'un obtient votre mot de passe, il peut lire vos emails, publier sur vos reseaux sociaux ou meme acceder a votre banque. La double authentification les arrete car ils auraient aussi besoin de votre telephone. C'est comme un deuxieme verrou sur votre porte d'entree.

### Etape 2 : Commencer par votre compte email

Votre email est le compte le plus important a proteger car les liens de reinitialisation de mot de passe y sont envoyes. Ouvrez le site de votre fournisseur d'email. Pour Gmail, allez sur **myaccount.google.com**, cliquez sur **Securite**, puis sur **Validation en deux etapes** et suivez les instructions. Pour les autres fournisseurs, cherchez **Securite** dans les parametres de votre compte.

### Etape 3 : Choisir votre methode de verification

On vous demandera comment vous souhaitez recevoir vos codes. Les deux options principales sont le **SMS** et l'**application d'authentification**. Choisissez **SMS** si vous voulez l'option la plus simple. Votre numero de telephone recevra un code par SMS a chaque connexion depuis un nouvel appareil.

### Etape 4 : Verifier votre numero de telephone

Entrez votre numero de telephone quand on vous le demande. Le service enverra un code test sur votre telephone par SMS. Ouvrez le message, lisez le code (generalement 6 chiffres) et tapez-le sur le site. Cela confirme que le numero est le votre et que vous pouvez recevoir des codes.

### Etape 5 : Sauvegarder vos codes de secours

Apres la configuration, le service vous montrera des **codes de secours**. Ce sont des codes a usage unique si vous perdez votre telephone. Ecrivez-les sur papier et gardez ce papier en lieu sur, comme dans un tiroir ou avec vos documents importants. Ne les sauvegardez pas uniquement sur votre telephone.

### Etape 6 : Proteger vos comptes de reseaux sociaux

Faites maintenant la meme chose pour vos autres comptes importants. Sur Facebook, allez dans **Parametres**, puis **Securite et connexion**, puis **Authentification a deux facteurs**. Sur WhatsApp, allez dans **Parametres**, puis **Compte**, puis **Verification en deux etapes**. Sur Instagram, allez dans **Parametres**, puis **Securite**, puis **Authentification a deux facteurs**.

### Etape 7 : Proteger vos comptes bancaires

Verifiez si votre banque propose la double authentification. La plupart des banques l'exigent deja pour la banque en ligne. Connectez-vous a votre application ou site bancaire, allez dans les **parametres de securite** et assurez-vous que la double authentification est activee. En cas de doute, appelez votre banque pour obtenir de l'aide.

## Conseils

- Commencez par votre compte email. Si un pirate accede a votre email, il peut reinitialiser les mots de passe de tous vos autres comptes.
- Si vous recevez un code de verification que vous n'avez pas demande, ne le partagez avec personne. Quelqu'un essaie peut-etre d'acceder a votre compte.
- Gardez votre numero de telephone a jour sur tous vos comptes. Si vous changez de numero, mettez-le a jour immediatement.
- Ecrivez vos codes de secours sur papier, pas dans une note sur votre telephone.
