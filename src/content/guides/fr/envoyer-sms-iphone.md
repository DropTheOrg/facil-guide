## Ce que vous allez apprendre

Dans ce guide, vous allez apprendre a envoyer un message texte sur votre iPhone avec l'application Messages. Vous pouvez l'utiliser pour envoyer un message rapide a votre famille, vos amis ou toute personne dans vos contacts.

## Ce dont vous avez besoin

- Un iPhone (n'importe quel modele)
- Le numero de telephone de la personne a qui vous voulez ecrire (ou elle doit etre dans vos contacts)

## Les etapes

### Etape 1 : Ouvrir l'application Messages

Trouvez l'application **Messages** sur votre ecran d'accueil. Elle a une icone verte avec une bulle de discussion blanche. Appuyez dessus.

### Etape 2 : Demarrer un nouveau message

Appuyez sur le **bouton de redaction** en haut a droite. Il ressemble a un petit carre avec un crayon. Cela ouvre un nouveau message vierge.

### Etape 3 : Entrer le destinataire

En haut de l'ecran, vous verrez un champ **A :**. Tapez le nom de la personne (si elle est dans vos contacts) ou son numero de telephone. Appuyez sur son nom quand il apparait dans les suggestions dessous.

### Etape 4 : Taper votre message

Appuyez sur le champ de texte en bas de l'ecran ou il est ecrit **iMessage** ou **Message texte**. Le clavier apparaitra. Tapez ce que vous voulez dire.

### Etape 5 : Appuyer sur le bouton d'envoi

Quand votre message est pret, appuyez sur le **bouton fleche bleue** a droite du champ de texte. Votre message sera envoye. Vous le verrez apparaitre dans une bulle bleue ou verte a l'ecran.

## Conseils utiles

- **Repondre a une conversation existante** : Si vous avez deja envoye un message a quelqu'un, trouvez son nom dans la liste Messages et appuyez dessus pour continuer la conversation.
- **Dictee vocale** : Si taper est difficile, appuyez longuement sur l'**icone microphone** du clavier et prononcez votre message. Le telephone le tapera pour vous.
- **Supprimer un message** : Appuyez longuement sur une bulle de message, puis appuyez sur **Plus**, selectionnez le message et appuyez sur l'icone corbeille pour le supprimer.
