## Project Summary

**facil.guide** is a multi-language how-to site for seniors. Static (Astro + Cloudflare Pages). Free hosting forever. Part of DropThe.

- **Stack:** Astro 5.x, static output, Cloudflare Pages
- **Languages:** FR (default), EN, ES, PT, IT
- **Content types:** Guides (step-by-step), Reviews (product), Comparisons (A vs B)
- **Categories:** Smartphone, Computer, Internet, Applications, Security, Communication, AI
- **Pages:** 87 (as of 2026-02-18)
- **Content:** 20 guides, 5 reviews, 5 comparisons (across 5 langs)
- **Build time:** ~600ms
- **Hosting cost:** $0
